1. The Neural Network folder contains the ML side of the project. It has the notebook where the model is trained along with preprocessing.

To run the application, simply download the folders as it is and the run the app.py file Downloads/Emotion-Based-E-Learning-Platform/Application/app.py and follow the link on any web browser.
